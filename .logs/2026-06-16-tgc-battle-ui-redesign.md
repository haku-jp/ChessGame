# TGC battle UI redesign

## Context

The mobile battle screen needed to stop showing a plain board under collection content and move toward a richer card-battle presentation.

## Changes

- Added a battle menu below the collection grid with standard deck start and battle preview actions.
- Added a dedicated full-screen battle scene so the collection screen no longer embeds a board.
- Rendered board pieces as circular cropped portraits from the card artwork.
- Rendered the player hand as a fanned row of mini cards inspired by physical card handling.
- Kept `tgc` and `docs/tgc` in sync for local preview and GitHub Pages output.

## Asset decision

Existing card files have distinct hashes, so no image generation was needed for this pass. Face-like board icons are generated by cropping the existing art with per-card focus positions.
