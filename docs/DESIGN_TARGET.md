# Visual Design Target (deferred reskin)

ループ完成後に着手するビジュアルリスキンのターゲット。HayattiQ提供のモックに準拠。
方針: **まず CSS 近似**（外部アセット不要）→ 後でスプライト差し替え。
`piece-glyph` は `<img>` に差し替え可能な構造を維持すること。

参照: GAME_SPEC §12（選択時表示）, §13（ビジュアル方針）。

## レイアウト骨格（段階①）
- 上部HUD: 金枠で区切ったセクション（ROUND / PHASE / COMMAND / SOUL）+ ロゴ
- COMMAND は数字ではなく **◆ピップ**（残数ぶん光る）
- SOUL に **🔥アイコン**
- 盤面: 黒曜石テクスチャ + 金の装飾枠 + 四隅の唐草
- **座標ラベル**（列 A–E / 行 1–5）
- 「ターン終了」は装飾枠ボタン + `Space` ホットキー表示

## リッチインスペクタ（段階②）
- 駒ポートレート（円形）+ 名前 + 陣営バッジ（光/闇）
- **HP を♥表示**（wounds → 残りハート）
- **XPバー**（level/xp）
- ステータス行: 攻撃（⚔ 1-2）/ 移動（L字移動）をアイコン付き
- 能力説明文
- **TRAITS** をタグ表示
- **移動可能ミニマップ**（movePattern を可視化、味方=金 / 敵取得=紫）

## bar・カウンター（段階③）
- 下部 **自軍ユニットbar**: 顔アイコン + HPバー（出撃/Bench UIの土台）
- 光/闇カウンター（盤上の駒数）
- 操作ヒント（左クリック: 選択/移動, 右クリック: キャンセル）

## 盤面質感（段階④）
- セルの黒曜石質感、選択=金リング、敵=紫グロー、移動先/取得先のハイライト調整

## 色（モックから）
- 背景: ほぼ黒〜濃紫 (#0c0a14 系)
- 金: #d9b066 / #f1cf85（縁・選択・味方リング）
- 闇陣営: 紫グロー (#6c4ab3 / #9b78e3)
- 光陣営: 象牙色の円盤 (#e9d8a6 + ハイライト)
- HP=赤♥, XP=シアン系バー, Soul=オレンジ🔥
