# TGC - 幻獣カード図鑑プロトタイプ

TGC は、子供の頃に考えた幻獣と魔法カードを集め、眺め、育て、進化させるカードゲーム案の静的 Web プロトタイプです。

Phase 1 では戦闘を作らず、カード図鑑、カード詳細、パック開封、localStorage による所持状態保存を優先しています。

## 起動方法

ビルドは不要です。

```text
tgc/index.html
```

をブラウザで直接開くか、任意の静的ファイルサーバーでリポジトリルートを配信してください。

## GitHub Pages URL

```text
https://haku-jp.github.io/ChessGame/tgc/
```

## 現在できること

- 所持カードと未所持カードを図鑑で確認
- カードタイプでフィルタ
  - すべて
  - 幻獣
  - オーラ
  - ソーサリー
  - アイテム
- カードクリックで詳細表示
- パック開封でランダムに 1 枚入手
- 入手済みカードは重複枚数として加算
- 所持状態を `localStorage` に保存
- 初回状態としてコランと光のオーラを所持
